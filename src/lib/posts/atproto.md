It's a schema for long-form publishing on [AT Protocol](https://atproto.com/). The idea is to keep the markdown notes in git as the source of truth and mirror them to atproto.

Looks like it's working :).

Next steps are CI (not sure if I want to further cement dependencies with Github Actions or steer towards selfhosting) and loading comments via atproto.
